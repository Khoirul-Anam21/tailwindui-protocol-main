# announcements

announcements are where the facilitator announce all the important information to its student. rather than in the comment, announcements should have a special and noticeable place {{ className: 'lead' }}

## The announcement model

The announcement model contains all the information about your announcements.

### Properties

<Properties>
  <Property name="id" type="string">
    Unique identifier for the announcement.
  </Property>
  <Property name="user_id" type="string">
    Unique identifier of user announcemented first.
  </Property>
  <Property name="course_id" type="string">
    Unique identifier of the course this announcement belongs to.
  </Property>
  <Property name="content" type="string">
    The content of the announcement.
  </Property>
  <Property name="chapter" type="integer">
    The chapter of the course announcemented on.
  </Property>
  <Property name="created_at" type="timestamp">
    Timestamp of when the announcement was created.
  </Property>
</Properties>

## List all announcements with reply {{ tag: 'GET', label: '/v1/announcements' }}

<Row>
  <Col>

    This endpoint allows you to retrieve a list of all your announcements.
    
    ### Required attributes

    <Properties>
      <Property name="course_id" type="string">
        The course of announcement referred.
      </Property>
    </Properties>

  </Col>

    <CodeGroup title="Request" tag="GET" label="/v1/announcements">

    ```bash {{ title: 'cURL' }}
    curl -G https://api.protocol.chat/v1/announcements \
      -H "Authorization: Bearer {token}" \
      -d course_id="csDaSDF44aas"
    ```

    ```js
    import ApiClient from '@example/protocol-api'

    const client = new ApiClient(token)

    await client.announcements.list({course_id:'csDaSDF44aas'})
    ```

    ```python
    from protocol_api import ApiClient
    ```
