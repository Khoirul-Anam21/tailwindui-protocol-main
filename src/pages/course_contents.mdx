# course_contents

Course content is the sub part of the course that contains the materials about the course. this is the part where users actually learn the course.

## The  model

The course_content model contains all the information about the course_contents.

### Properties

<Properties>
  <Property name="id" type="string">
    Unique identifier for the course_content.
  </Property>
  <Property name="course" type="object">
    The course information containing this content.
  </Property>
  <Property name="title" type="string">
    The title of course content.
  </Property>
  <Property name="description" type="string">
    The description or content material.
  </Property>
  <Property name="source" type="string">
    A Url for participant's submission / supporting file.
  </Property>
  <Property name="assignment" type="object">
    Instruction for giving task / assignment for participants.
  </Property>
    <Property name="announcement" type="array">
    List of the course announcement.
  </Property>
    <Property name="video" type="string">
    Url of video course content.
  </Property>
  <Property name="created_at" type="timestamp">
    Timestamp of when the course_content was created.
  </Property>
  <Property name="updated_at" type="timestamp">
    Timestamp of when the course_content was updated.
  </Property>
</Properties>

## List all course_contents {{ tag: 'GET', label: '/v1/course_contents' }}

<Row>
  <Col>

    ### Required attributes

    <Properties>
      <Property name="course_id" type="string">
        Get all course content in the specific course.
