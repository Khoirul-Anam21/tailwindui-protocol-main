# Courses

Course is the main product of this application. Course is the topic of the learning that contains many contents{{ className: 'lead' }}

## The course models

The course model contains all the information about the courses. it contains of the information of facilitator data, title, category, thumbnail, description.

### Properties

<Properties>
  <Property name="id" type="string">
    Unique identifier for the course.
  </Property>
  <Property name="facilitator" type="string">
    The facilitator of this course.
  </Property>
  <Property name="title" type="string">
    The title of the course.
  </Property>
  <Property name="category_id" type="string">
    Unique identifier for category of the course.
  </Property>
  <Property name="thumbnail" type="string">
    The thumbnail of the course.
  </Property>
  <Property name="description" type="string">
    The description of the course.
  </Property>
  <Property name="created_at" type="timestamp">
    Timestamp of when the course was created.
  </Property>
  <Property name="updated_at" type="timestamp">
    Timestamp of when the course was updated.
  </Property>
</Properties>

## List all courses {{ tag: 'GET', label: '/v1/courses' }}

<Row>
  <Col>

    This endpoint allows you to retrieve a paginated list of all your courses. This endpoint usually used for catalog. By default, a maximum of ten courses are shown per page.

    ### Optional attributes

    <Properties>
      <Property name="limit" type="integer">
        Limit the number of courses returned.
      </Property>
      <Property name="category_id" type="string">
        Filter courses by their categories.
      </Property>
    </Properties>

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="GET" label="/v1/courses">

    ```js
    import ApiClient from '@example/protocol-api'

    const client = new ApiClient(token)

    await client.courses.list()
    ```

    </CodeGroup>
