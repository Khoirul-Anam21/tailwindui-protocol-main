# Dashboard Report

reports are how student or facilitator analize or obtain information about their activity in this LMS {{ className: 'lead' }}

## Student report model 

The report model contains all the learning informations and progresses.

### Properties

<Properties>
  <Property name="enrolledCoursesTotal" type="number">
    Total course enrolled
  </Property>
  <Property name="completedCourses" type="number">
    total course completed
  </Property>
  <Property name="assignmentDue" type="number">
    total assignment not done
  </Property>
  <Property name="enrolledCourses" type="array">
    List of enrolled courses
  </Property>
  
</Properties>

## Facilitator report model

The report model contains all the information about courses informations that the facilitator has.

### Properties

<Properties>
  <Property name="ownedCourseTotal" type="number">
    Total course owned
  </Property>
  <Property name="assignmentCreated" type="number">
    total assignment created
  </Property>
  <Property name="participantTotal" type="number">
    total participant enrolled
  </Property>
  <Property name="ownedCourse" type="array">
    List of owned courses
  </Property>
  <Property name="participantEngagement" type="object">
    Periodic data of participant enrollment
  </Property>
</Properties>
