# reports

reports are how you share things in Protocol — they allow you to send all sorts of files to your contacts and groups. On this page, we'll dive into the different report endpoints you can use to manage reports programmatically. We'll look at how to query, upload, update, and delete reports. {{ className: 'lead' }}

## Student report model 

The report model contains all the information about the files you send to your contacts and groups, including the name, type, and size.

### Properties

<Properties>
  <Property name="id" type="string">
    Unique identifier for the report.
  </Property>
  <Property name="user_id" type="string">
    Unique identifier for the user associated with the report.
  </Property>
  <Property name="course_id" type="string">
    Unique identifier for the course associated with the report
  </Property>
  <Property name="progress" type="double">
    The percentage of course completion.
  </Property>
  <Property name="score" type="string">
    The score report of the course.
  </Property>
  <Property name="status" type="integer">
    The completion status of the course.
  </Property>
  <Property name="latest_update" type="timestamp">
    Timestamp of when the report was updated.
  </Property>
</Properties>

## Get participant report by id {{ tag: 'GET', label: '/v1/reports/participants/:id' }}

<Row>
  <Col>

    This endpoint allows you to retrieve a paginated list of all your reports (in a conversation if a conversation id is provided). By default, a maximum of ten reports are shown per page.


  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="GET" label="/v1/reports">

    ```bash {{ title: 'cURL' }}
    curl -G https://api.protocol.chat/v1/reports/participants/xdagSd64 \
      -H "Authorization: Bearer {token}" \
    ```

    ```js
    import ApiClient from '@example/protocol-api'

    const client = new ApiClient(token)

    await client.reports.participants.get("xdagSd64")
    ```

    ```python
    from protocol_api import ApiClient

    client = ApiClient(token)

    client.reports.participants.get("xdagSd64")
    ```

    ```php
    $client = new \Protocol\ApiClient($token);

    $client->reports->participants->get("xdagSd64");
    ```
